# Security Policy

## Phiên bản được hỗ trợ

nonlaOS đang ở giai đoạn alpha/pre-release. Chưa có bản stable được hỗ trợ dài
hạn.

## Báo cáo lỗ hổng

Không mở public issue cho lỗ hổng có thể bị khai thác ngay.

Hãy dùng GitHub Security Advisory của repo nếu có quyền truy cập, hoặc liên hệ
maintainer qua kênh riêng của dự án.

Khi báo cáo, vui lòng gửi:

- mô tả vấn đề
- bước tái hiện
- package hoặc file bị ảnh hưởng
- mức độ ảnh hưởng dự kiến
- bản vá gợi ý nếu có

## Phạm vi

Phạm vi bảo mật hiện tại gồm:

- Debian packaging của nonlaOS
- cấu hình hệ thống do package nonlaOS ship
- script build trong repo
- asset/theme có thể ảnh hưởng login, boot hoặc desktop shell
