# Support

nonlaOS hiện là dự án đang phát triển, chưa phải bản phân phối ổn định cho
người dùng cuối.

## Hỏi đáp

Dùng GitHub Discussions nếu repo bật Discussions. Nếu chưa bật, dùng GitHub
Issues với template phù hợp.

## Báo lỗi

Khi báo lỗi, vui lòng kèm:

- phiên bản commit hoặc package
- môi trường test: Debian/WSL/VM/máy thật
- log lệnh build hoặc lint
- bước tái hiện ngắn gọn

## Không hỗ trợ

Dự án không hỗ trợ:

- phần mềm lậu, crack, bypass license
- ISO bị sửa thủ công ngoài quy trình build
- asset không rõ nguồn/license
